  ```
  normFluidRisk('nivel optimo, sin fuga') // => 'Urgente'  (front/risk.ts => 'OK')
  normFluidRisk('sin fuga')                // => 'Urgente'
  normFluidRisk('no hay fuga')             // => 'Urgente'
  ```
- **Fix:** Reordenar como en risk.ts/HTML (evaluar OK antes que URG):
  ```js
  function normFluidRisk(val) {
    const v = norm(val);
    if (!v) return "OK";
    if (OK.some((kw) => v === kw || v.includes(kw))) return "OK";
    if (v === "si") return "OK";
    if (URG.some((kw) => v.includes(kw))) return "Urgente";
    return "Revisar";
  }
  ```
  Mover el bloque OK (L279-319) antes del bloque URG (L263-278). **Backfill de semanales ya ingeridas.**
